HOWTO - Deploy

All scripts can be started with the external tools menu in Eclipse or manually on the command line.

-- Quick start to compile an exe ---------------------------------------------------------------------

	1. Get xampp.zip from: \\Link3\projekte\HALO-Vulcan\02 HALO-Vulcan-Refinement Phase\Software_Backup\xampp
	2. Copy it to extensions/SMWHalo/deploy/bin
	3. Run Javascript CopyScripts
	4. Run Javascript Obfuscator (optionally, can be skipped. Needs Jasob: www.jasob.com)
	5. Run Javscript Packer
	6. Run makeAll
	7. Run makeExecutable
	
Then you'll find halowiki.exe in extensions/SMWHalo/deploy/bin

Have fun!

Javascript changes.

Changes in a javascript file need special treatment. Otherwise the deployversion of the scripts
will not work. These steps must be done *before* any of the make command from above is executed.

 1. Start 'Javascript CopyScripts' 
 2. Start 'Javascript Obfuscator' (needs Jasob installed: www.jasob.com, but can be skipped)
 3. Start 'Javascript Packer'
 
Refresh workspace, you should see some new deploy*.js files.
